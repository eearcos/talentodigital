PROYECTO: GESTOR DE BIBLIOTECA
==============================

DESCRIPCIÓN:
Sistema de gestión de biblioteca desarrollado en Python utilizando Programación Orientada a Objetos (POO).
Permite administrar libros físicos y digitales, gestionando su estado (disponible/prestado) y persistiendo
la información en un archivo de texto.

REQUISITOS:
- Python 3.x instalado.

ARCHIVOS DEL PROYECTO:
1. gestor_biblioteca.py: Código fuente principal.
2. biblioteca.txt: Base de datos de texto (se crea automáticamente).
3. README.txt: Este archivo de instrucciones.

INSTRUCCIONES DE USO:
1. Abrir la terminal o línea de comandos.
2. Navegar hasta la carpeta donde se encuentran los archivos.
3. Ejecutar el comando: 
   python gestor_biblioteca.py
4. Seguir las instrucciones del menú numérico en pantalla.

FUNCIONALIDADES:
- Agregar libros (Físicos o Digitales).
- Listar inventario.
- Buscar libros por título.
- Prestar y devolver libros (cambio de estado).
- Eliminar libros.
- Guardado automático de datos al realizar cambios.
