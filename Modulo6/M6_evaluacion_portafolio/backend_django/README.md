# Proyecto Django - Growth Academy

## Características de Django (Investigación)
Django es un framework web de alto nivel para Python que fomenta el desarrollo rápido y el diseño limpio. Sus ventajas para aplicaciones empresariales incluyen:
1.  **"Baterías incluidas":** Trae autenticación, panel de administración, y manejo de formularios listos para usar.
2.  **Seguridad:** Protege automáticamente contra ataques comunes como SQL Injection y CSRF.
3.  **Escalabilidad:** Su arquitectura MVT (Modelo-Vista-Template) permite separar la lógica de negocio de la interfaz visual, facilitando el mantenimiento.
4.  **ORM:** Permite interactuar con la base de datos usando objetos Python en lugar de escribir SQL puro.

## Estructura del Proyecto
- `gestion_academica`: Configuración global.
- `core`: Aplicación principal que gestiona Cursos y Estudiantes.
